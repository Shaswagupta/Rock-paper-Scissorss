# Rock Paper Scissors Game

A simple, clean, and modern Rock Paper Scissors web game built with HTML, CSS, and JavaScript.

## Features

- 🎮 Interactive gameplay with computer opponent
- 📊 Real-time scoreboard
- 🎨 Modern, responsive UI
- 📱 Works on mobile and desktop
- ♿ Beginner-friendly code

## How to Play

1. Click Rock, Paper, or Scissors
2. Computer makes a random choice
3. Result is displayed with score updated
4. Click "Reset Score" to start over

## Files

- `index.html` - HTML structure
- `style.css` - Styling and responsiveness
- `script.js` - Game logic
